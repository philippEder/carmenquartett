# CarmenQuartett

This is a Angular application for the Carmen Quartett website.

## How to run locally

1. Install Node.js (16+ recommended) and npm.
2. From the project root (where package.json is), run:
   ```bash
   npm install
   npm start
   ```

3. The app will open at http://localhost:4200
